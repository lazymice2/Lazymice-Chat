# Lazymice Chat

#### 介绍
该项目为一个包含多种部署方式的集成AIGC平台生态链，拥有集合ChatGPT和Midjourney的网站Web应用。
<br> ✅可以部署基于微信的机器人bot，支持在微信端部署机器人进行AI行为。
<br> ✅提供ChatGPT、Midjourney在内的多个平台的转发api，代理地址，方便小白用户直接在国内环境进行项目部署。
<br> ✅提供API分销、微信机器人代理、网站部署等多种服务。
